# silk2wav

Convert wechat audio files (Skype silk format) to wav format

# Build

```bash
# build
make
# clean
make clean
```

Tested under

| Sys     | Checked |
|---------|---------|
| MacOS X | ✅       |
| Ubuntu  | ✅       |
| CentOS  | ✅       |
| Windows | Todo    |

# Usage

```bash
./bin/wechat2wav_macos inputfile outputfile
```
